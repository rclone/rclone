# {{< icon "fa fa-folder" >}} NetExplorer

[NetExplorer](https://www.netexplorer.fr/) is a secure file sharing and
collaboration platform for enterprises.

## Configuration

Here is an example of how to configure a NetExplorer remote called `remote`.

Start with:

```sh
rclone config
```

Then follow the prompts:

```text
No remotes found, make a new one?
n) New remote
s) Set configuration password
q) Quit config
n/s/q> n

Enter name for new remote.
name> remote

Option Storage.
Type of storage to configure.
Choose a number from below, or type in your own value.
XX / NetExplorer remote
   \ (netexplorer)
Storage> netexplorer

Option base_url.
Base URL of the NetExplorer API
Enter a string value. Press Enter for the default.
base_url> https://org.netexplorer.pro

Option token.
Access token for NetExplorer
Enter a string value. Press Enter to leave empty.
token>

Option email.
Your NetExplorer user email (used to fetch a token if you don't paste one)
Enter a string value. Press Enter to leave empty.
email> user@example.com

Option password.
Your NetExplorer password (used to fetch a token if you don't paste one)
Enter the password:
password:
Confirm the password:
password:
```

If `token` is left empty and `email`/`password` are provided, rclone
will request a token automatically.

## Root Folder

By default, rclone uses the root workspace exposed by the NetExplorer API.
You can optionally restrict the remote to a specific root folder with the
`root` option (numeric folder ID).

## Notes

- A token is preferred for automation.
- If credentials are configured, the backend can re-authenticate on HTTP 401.
- The backend validates names and rejects forbidden characters
  (`\ / : * ? " < > |`).

### Standard options

Here are the Standard options specific to netexplorer (NetExplorer remote).

#### --netexplorer-base-url

Base URL of the NetExplorer API

Properties:

- Config:      base_url
- Env Var:     RCLONE_NETEXPLORER_BASE_URL
- Type:        string
- Default:     "https://org.netexplorer.pro"

#### --netexplorer-email

Your NetExplorer user email (used to fetch a token if you don't paste one)

Properties:

- Config:      email
- Env Var:     RCLONE_NETEXPLORER_EMAIL
- Type:        string
- Required:    false

#### --netexplorer-password

Your NetExplorer password (used to fetch a token if you don't paste one)

**NB** Input to this must be obscured - see [rclone obscure](/commands/rclone_obscure/).

Properties:

- Config:      password
- Env Var:     RCLONE_NETEXPLORER_PASSWORD
- Type:        string
- Required:    true
