# {{< icon "fa fa-video" >}} Darkibox

[Darkibox](https://darkibox.com/) is a video hosting platform based on
the XFileSharing framework. It provides online storage and video
streaming capabilities.

## Configuration

Here is an example of how to make a remote called `remote`. First run:

    rclone config

This will guide you through an interactive setup process:

```
No remotes found, make a new one?
n) New remote
s) Set defaults
q) Quit config
n/s/q> n
name> remote
Type of storage to configure.
Choose a number from below, or type in your own value
XX / Darkibox
   \ "darkibox"
Storage> darkibox
API key for your darkibox account.
Found on https://darkibox.com/api.html when logged in.
api_key> YOUR_API_KEY_HERE
--------------------
[remote]
type = darkibox
api_key = YOUR_API_KEY_HERE
--------------------
y) Yes this is OK
e) Edit this remote
d) Delete this remote
y/e/d> y
```

### Getting your API key

1. Log in to your account on [darkibox.com](https://darkibox.com/)
2. Go to the [API page](https://darkibox.com/api.html)
3. Copy your API key

### Usage

List directories in top level of your remote:

    rclone lsd remote:

List all the files in your remote:

    rclone ls remote:

Upload a video file:

    rclone copy /path/to/video.mp4 remote:folder/

Download a video file:

    rclone copy remote:folder/video.mp4 /path/to/local/

### Limitations

- Darkibox only accepts video files. Non-video uploads will be rejected.
- Video files must be at least 10 seconds long.
- File modification times cannot be set via the API.
- Hash checking is not supported.
- The API has a rate limit on requests.

### Standard options

Here are the Standard options specific to darkibox (Darkibox).

#### --darkibox-api-key

API key for your darkibox account.

Found on https://darkibox.com/api.html when logged in.

Properties:

- Config:      api_key
- Env Var:     RCLONE_DARKIBOX_API_KEY
- Type:        string
- Required:    true

### Advanced options

Here are the Advanced options specific to darkibox (Darkibox).

#### --darkibox-api-url

API endpoint URL.

Normally this doesn't need to be changed.

Properties:

- Config:      api_url
- Env Var:     RCLONE_DARKIBOX_API_URL
- Type:        string
